### Riemann Zeta Function

```python
import numpy as np

def riemann_zeta(s):
    """
    Compute the Riemann zeta function at s.

    Parameters:
    s (float): The complex number at which to compute the zeta function.

    Returns:
    float: The value of the zeta function at s.
    """
    s = np.complex128(s)
    if s == 1:
        raise ValueError("Zeta function is undefined at s = 1")
    return 1 / np.prod(1 - 1 / (np.arange(1, 10000) + 1) ** s)
```

### Non-Trivial Zeros

```python
import numpy as np

def non_trivial_zeros(n):
    """
    Compute the first n non-trivial zeros of the Riemann zeta function.

    Parameters:
    n (int): The number of non-trivial zeros to compute.

    Returns:
    list: A list of the first n non-trivial zeros.
    """
    zeros = []
    for i in range(n):
        s = 1/2 + 1j * np.log(i) / np.log(10)
        if np.isclose(riemann_zeta(s), 0, atol=1e-10):
            zeros.append(s)
    return zeros
```

### Functional Equations

```python
import numpy as np

def functional_equation(s):
    """
    Compute the functional equation of the Riemann zeta function at s.

    Parameters:
    s (float): The complex number at which to compute the functional equation.

    Returns:
    float: The value of the functional equation at s.
    """
    s = np.complex128(s)
    return 2 * (1 - 2**(-s)) * riemann_zeta(s)
```

**API Endpoints**
----------------

### GET /zeta

Compute the Riemann zeta function at s.

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/zeta', methods=['GET'])
def get_zeta():
    s = float(request.args.get('s'))
    return jsonify({'zeta': riemann_zeta(s).real})
```

### GET /zeros

Compute the first n non-trivial zeros of the Riemann zeta function.

```python
@app.route('/zeros', methods=['GET'])
def get_zeros():
    n = int(request.args.get('n'))
    return jsonify({'zeros': [z.real for z in non_trivial_zeros(n)]})
```
