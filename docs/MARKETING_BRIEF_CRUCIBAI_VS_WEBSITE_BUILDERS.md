# CrucibAI Marketing Brief: Website Builders And AI Product Builders

Last updated: 2026-05-03

## Positioning

CrucibAI is not positioned as a static website builder. It is an AI product-building platform for apps, automations, previews, proof, and deployment paths.

Approved headline:

> Build apps and automations from one prompt, with proof and deployment paths built in.

Approved subhead:

> CrucibAI combines a workspace, agent execution, PayPal billing, cost governance, previews, and trust endpoints so users can move from idea or imported code to a working product path.

## Evidence To Use

| Claim | Evidence |
| --- | --- |
| App and automation platform | Workspace, jobs, automation, simulations, preview, and published app routes. |
| Railway production path | `railway.json`, `Dockerfile`, `scripts/railway_release_smoke.py`. |
| Repeatability gate | `benchmarks/repeatability_prompts_v1.json`, `backend/tests/test_repeatability_benchmark.py`. |
| Billing | PayPal billing route and PayPal billing tests. |
| Cost governance | `/api/cost/governance`. |
| Trust | `/api/trust/enterprise-readiness`, `/api/trust/public-proof-readiness`, `/api/trust/summary`. |

## Competitive Framing

Use:

> Website builders help you publish pages. CrucibAI is built to create product workflows: app structure, automation logic, proof, preview, billing, and deployment readiness.

Avoid:

> CrucibAI is guaranteed to beat every website builder.

## Claims Held For Gate Approval

The following claims are not approved for public use until the current release has timestamped proof:

- "99.2 percent guaranteed success."
- "Outcome guaranteed."
- "#1 AI coding platform."
- "Better than all competitors."
- "Fortune 500 guaranteed."

## Call To Action

Use:

> Tell CrucibAI what you want to build. It will plan, build, verify, preview, and prepare the product path.
