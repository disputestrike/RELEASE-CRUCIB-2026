# Your logo — icon + name together

The app shows **your logo icon** and the name **"CrucibAI"** side by side everywhere. No custom graphic is used; only your asset.

## What to add

1. **Your logo icon** (the dark rounded square with the 2×2 grid)  
   - Save it as: **`logo-icon.png`**  
   - Put it in: **`frontend/public/`**  
   - So the file is: **`frontend/public/logo-icon.png`**

The UI will load this image and display it next to the text "CrucibAI". If the file is missing, only "CrucibAI" text is shown until you add it.

## Optional (for favicon / PWA)

- **`logo-favicon.ico`** — browser tab
- **`logo-icon-512.png`** — Apple touch / PWA (can be the same icon at 512×512)

You can add your icon again or a higher‑res version; we use only what you provide and do not create or replace it.
